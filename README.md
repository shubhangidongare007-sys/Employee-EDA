# Employee EDA

## Overview
This project performs Exploratory Data Analysis (EDA) on an Employee Dataset using Python.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- Employee Dataset (employee.csv)

## Tasks Performed
- Loaded the dataset
- Displayed the first 10 rows
- Checked dataset shape and information
- Generated statistical summary
- Checked missing values
- Identified numerical and categorical columns
- Created distribution plots
- Created count plots
- Generated correlation heatmap
- Performed salary analysis
- Created boxplots
- Summarized key findings

## Output
The project includes Python code, visualizations, and analysis of the Employee dataset.
